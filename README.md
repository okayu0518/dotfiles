# Dotfiles - Keep It Simple

シンプルで効率的な開発環境設定

## インストール

```bash
git clone https://github.com/okayu0518/dotfiles.git ~/dotfiles
cd ~/dotfiles
chmod +x install.sh
./install.sh
```

## 含まれる設定

- **Neovim**: ミニマルで高速な設定
- **Emacs**: init.el設定
- **Terminal**: Alacritty, Wezterm, Bash, Tmux
- **Git**: 基本設定
- **Font**: Hack Nerd Font

## キーバインド

### Neovim
- `<leader>ff`: ファイル検索
- `<leader>fg`: 文字列検索
- `<leader>fb`: バッファ一覧
- `Alt+j/k`: 行移動
- `Ctrl+hjkl`: ウィンドウ移動

### システム
- `CapsLock`: Ctrl に変更
